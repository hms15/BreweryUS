    Hybrid Structure

    This project is based on the interactive slideshow method. The user will be navigating through several scenes learning about breweries and beer. The interactive feature entails hover events.

    Scenes

    From the home page users will be able to start the slideshow. They will be directed to a slideshow created by using Bootstrap's scrolling navigation template.

    Annotations

    I have utilized a hover mechanism to support annotations which will highlight data. The annotation will disappear when the user moves the mouse or changes screens.

    Parameters

    I have used aggregated values in my charts such as beer style counts, beer counts, and counts of breweries per state.

    Triggers

    A scrolling mechanism is implement to seamlessly navigate through the slides.
